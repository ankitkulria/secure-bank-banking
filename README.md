# Secure Bank Banking System

A full-stack Banking Management System built using Spring Boot, MySQL, JWT Authentication, HTML, CSS, JavaScript and Bootstrap.

## Project Structure

secure-bank-banking/

├── BankManagementSystem (Backend)

└── BankingFrontend (Frontend)

## Features

- JWT Authentication
- Account Management
- Deposit & Withdraw
- Fund Transfer
- Transaction History
- Loan Management
- EMI Payments
- Admin Dashboard
- Email Notifications

## Tech Stack

Backend:
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL

Frontend:
- HTML
- CSS
- JavaScript
- Bootstrap
